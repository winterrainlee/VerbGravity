# Release Notes - v1.1.2

**Release Date**: 2026-01-20

## 🚀 주요 변경 사항

### 1. 학습자 중심의 UI/UX 개선
- **주어구 힌트 정교화**: 기초 모드에서 주어 힌트 표시 범위를 핵심 단어(Head)에서 전체 주어구(Span)로 확대하여 학습자가 주어의 범위를 명확히 파악할 수 있도록 도왔습니다.
- **채점 모드 상태 배지 추가**: 시작 화면에서 현재 설정된 채점 모드(기초/심화)를 배지 형태로 노출하여 학습 시작 전 상태를 인지할 수 있도록 개선했습니다.
- **모드별 안내 메시지 강화**: 퀴즈 화면의 모드 배지 클릭 시, 각 모드의 핵심 원칙(기초 1개 / 심화 다수 등)을 설명하는 맞춤형 안내를 제공합니다.

### 2. NLP 분석 엔진 고도화
- **모드별 분석 원칙 수립**: 
  - **기초 모드**: 문장당 단 하나의 메인 뿌리동사와 주어만 정답으로 도출합니다.
  - **심화 모드**: 부사절, 보문절 등 문장 내 모든 핵심 동사를 정답으로 포함합니다.
- **관계절(Relative Clauses) 정밀도 향상**: `who`, `which`, `that` 관계절 문장에서 주어구(Span)를 추출할 때 관계절 내부 요소를 수식어로 분리하여 제외하는 로직을 추가했습니다.
- **부사절(Adverbial Clauses) 지원**: `because`, `if`, `when` 등이 이끄는 부사절 동사를 뿌리동사(Root) 후보에 포함시켜 더 복합적인 문장 훈련이 가능해졌습니다.

### 3. 시스템 안정성
- **NLP 오류 트래킹 시스템**: 분석 실패 시 상세 정보(문장, 에러 타입)를 기록하는 별도 로그(`nlp_errors.log`) 기능을 추가하여 후속 개선의 기반을 마련했습니다.

---

## ✅ 검증 요약
- **Pytest**: 관계절/부사절 및 모드별 필터링 테스트 5종 전원 통과.
- **Manual Verification**: 복합 문장(because 절 포함)을 이용한 기초/심화 모드 동작 차이 최종 확인 완료.
