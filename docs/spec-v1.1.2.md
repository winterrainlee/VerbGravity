# 버전 명세서 (Specification) - v1.1.2

## 개요
- **목표**: 사용자 경험 개선 및 NLP 분석 안정성 향상
- **유형**: Patch
- **예상 기간**: 1일

---

## 기능 목록

### 1. 주어구 힌트 표시 수정
- **설명**: 기초 모드에서 핵심동사 맞춘 후 주어구(Subject Span) 전체에 밑줄 표시
- **현재 동작**: 핵심 주어(Head)만 밑줄
- **목표 동작**: 주어구 전체(예: "The quick brown fox")에 밑줄
- **영향 범위**: Frontend
- **관련 파일**: `QuizScreen.jsx`

### 2. 시작화면 모드 표시
- **설명**: 현재 채점 모드를 시작화면에 읽기 전용으로 표시
- **표시 형식**: 🟢 기초 모드 / 🟠 심화 모드
- **영향 범위**: Frontend
- **관련 파일**: `StartScreen.jsx`, `StartScreen.css`

### 3. NLP 관계절 처리 개선
- **설명**: who/which/that 관계절에서 주어 인식 정확도 향상
- **영향 범위**: Backend
- **관련 파일**: `analyzer.py`

### 4. NLP 오류 분석 리포트
- **설명**: NLP 분석 실패 케이스 수집 및 로깅 체계 구축
- **영향 범위**: Backend
- **관련 파일**: `error_tracker.py` (신규)

---
