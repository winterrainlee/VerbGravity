# Release Notes - v1.0.0 (MVP)

> **Release Date:** 2026-01-16
> **Version:** v1.0.0 (First Official Release)

VerbGravity의 첫 번째 공식 버전인 v1.0.0을 출시합니다! 
본 버전은 영어 문장의 핵심 구조(동사와 주어)를 파악하는 핵심 학습 루프와 교사를 위한 관리 시스템을 갖춘 MVP(Minimum Viable Product) 모델입니다.

## ✨ 새로운 기능

### 1. 학생 학습 모드 (Student Mode)
- **지능형 NLP 퀴즈**: 입력한 지문을 spaCy를 이용해 실시간 분석하여 동사와 주어를 찾는 문제로 변환합니다.
- **단계별 인터랙션**: 
  - **1단계 (Root)**: 문장의 핵심 동사 찾기
  - **2단계 (Subject)**: 주어 핵(Head) 찾기
- **실시간 피드백**: 오답 시 품사 정보를 기반으로 한 힌트를 제공하여 스스로 교정할 수 있도록 돕습니다.
- **오답 복습 시스템**: 학습 종료 후 틀린 문장만 다시 풀어보는 복습 모드를 지원하며, 완료 시 축하 메시지를 출력합니다.

### 2. 교사 관리 모드 (Teacher Mode)
- **관리자 로그인**: 비밀번호 인증을 통해 교사용 대시보드에 접속할 수 있습니다.
- **학습 통계 대시보드**: 학생별 정답률(동사/주어), 완료 문장 수, 최근 학습 시간을 한눈에 보여줍니다.
- **세션 및 학생 관리**:
  - 생성된 학습 세션을 학생 이름으로 배정하여 관리할 수 있습니다.
  - 불필요한 학생 데이터를 삭제하여 관리 효율성을 높였습니다.

### 3. 시스템 및 인프라
- **학습 데이터 유지 (Persistence)**: SQLite 기반 백엔드를 통해 새로고침이나 재접속 시에도 학습 데이터를 복원합니다.
- **보안 및 안정성**: IP당 API 요청 제한(Rate Limiting)과 관리자 인증 로직을 적용했습니다.
- **최적화된 아키텍처**: 커스텀 훅(`useQuiz`)과 전역 상태 관리(`QuizContext`)를 도입하여 빠르고 안정적인 사용자 경험을 제공합니다.
